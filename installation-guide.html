Setup Instructions

Follow these steps to load the extension manually into your Chromium-based browser.

01

Download Package

Download the extension .zip file from the main page and extract it to a permanent folder on your drive.

02

Access Extensions

Open your browser and navigate to chrome://extensions (works for Brave and Edge as well).

03

Enable Developer Mode

Locate the Developer Mode toggle in the top-right corner and switch it to the **ON** position.

04

Load Unpacked

Click the Load unpacked button and select the folder where you extracted the extension files.

Running into issues? Ensure you are using a Chromium-based browser like Chrome, Brave, or Edge.
